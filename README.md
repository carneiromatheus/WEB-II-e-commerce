# WEB II: e-commerce
Trabalho realizado para a disciplica de WEB II do curso de Sistema de Informações do IFMG Sabará

# Tecnologias utilizadas

- Node.js 20^;
- Express;
- PostegreSQL;
- Handlebars.

## Como executar

1. Clone este repositório

```bash
git clone git@github.com:carneiromatheus/WEB-II-e-commerce.git

# Ou

git clone https://github.com/carneiromatheus/WEB-II-e-commerce.git
```

2. Instale todas as dependências

```bash
npm i
```

3. Crie e configure o arquivo `.env` usando o arquivo exemplo `.env.sample`.
4. Execute a aplicação em modo de desenvolvimento:

```bash
npm run dev
```
